# Schema

This folder contains simple starter JSON Schema files for the Age Calculator project.

## Files

- `birth-date.schema.json` — validates a user's date-of-birth input.
- `age-result.schema.json` — validates calculated age output.
- `age-calculation.schema.json` — validates a complete age calculation request/response record.
- `index.schema.json` — collection-level schema that references the other schemas.

These schemas use JSON Schema Draft 2020-12 and can be expanded as the project grows.
